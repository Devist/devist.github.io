2019-01-15
esdoc, jsdoc

esdoc은 JavaScript 용 문서 제네레이터, 특히 es6에 특화된 documentation generator입니다. 비슷하게 JavaScript의 주석을 문서로 만들어주는 것이로, JSdoc 도 있습니다.

현재 진행하고 있는 프로젝트에는 esdoc을 적용해보려고 합니다.

적용하려는 프로젝트는 다음과 같이 구성되어 있습니다.

  • Webpack
  • Javascript (es6)
